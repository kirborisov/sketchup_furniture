# Мебельный Конструктор для SketchUp

Модульная система для создания корпусной мебели в SketchUp с автоматическим расчётом раскроя и фурнитуры.

## Возможности

- ✅ Создание шкафов, гардеробов, встроенных систем
- ✅ Автоматический расчёт раскроя (ЛДСП, ДВП, фанера)
- ✅ Список фурнитуры
- ✅ Секции с процентным соотношением
- ✅ Полки фиксированные и регулируемые
- ✅ **Ящики** с направляющими и анимацией
- ✅ **Цоколь** — боковины до пола, дно поднято
- ✅ **Ножки** — боковины короче, с визуализацией
- ✅ **Пропуск частей** — без дна, без задника
- ✅ **Анимация** — открытие/закрытие ящиков
- ✅ **Двойной клик** — открытие ящиков и дверей мышкой (DrawerTool)
- ✅ **Ряды ящиков** — несколько ящиков по горизонтали (drawer_row)
- ✅ **Царги** — стандартные и под мойку (stretchers)
- ✅ **Кухня** — нижний/верхний ряд, столешница (Kitchen)
- ✅ **Столешница** — с выступом (countertop)
- ✅ **Двери** — одна или две створки, вращение по петлям (doors 1 / doors 2)
- ✅ **Рамочные двери** — рамка из массива с шипом и филёнка из фанеры (type: :frame)


## Полный пример: Прихожая

```ruby
wardrobe = Wardrobe.new "Прихожая", depth: 400 do
  
  # ═══════════ ЛЕВАЯ КОЛОННА ═══════════
  column 900 do
    
    # Низ — обувница (с цоколем 80мм)
    base 450, name: "Обувь" do
      plinth 80                 # ниша 80мм снизу
      sections "50%", "50%"
      shelf 150
    end
    
    # Середина — одежда
    cabinet 1450, name: "Одежда" do
      sections "50%", "50%"
      shelf 900
    end
    
    # Антресоль
    top 800, name: "Антресоль", shelf: 400
  end
  
  # ═══════════ ЦЕНТРАЛЬНАЯ КОЛОННА ═══════════
  column 700 do
    
    # Низ — ящики
    base 450, name: "Ящики" do
      skip :bottom              # дно не нужно
      drawers 3, height: 140, slide: :ball_bearing
    end
    
    # Середина — полки
    cabinet 1450, name: "Полки" do
      sections "50%", "50%"
      shelves [300, 600, 900, 1200]
    end
    
    # Антресоль
    top 800, name: "Антресоль", shelf: 400
  end
  
  # ═══════════ СКАМЕЙКА ═══════════
  column 1000 do
    
    # Скамейка на ножках
    base 450, name: "Скамейка" do
      legs 100                  # ножки 100мм
      shelf 150
    end
    
    # Над скамейкой — открытое (крючки)
    cabinet 1450, name: "Крючки" do
      skip :back                # без задника — стена видна
    end
    
    # Антресоль
    top 800, name: "Антресоль", shelf: 400
  end
  
end

# Построить
wardrobe.build

# Результаты
wardrobe.summary
wardrobe.print_cut_list
wardrobe.print_hardware_list

# Анимация ящиков
wardrobe.open_all_drawers      # открыть все
wardrobe.close_all_drawers     # закрыть все
```

## Ящики

### Добавление ящиков

```ruby
cabinet 500 do
  # Один ящик
  drawer 150
  
  # Несколько одинаковых
  drawers 3, height: 140
  
  # С параметрами
  drawer 200, slide: :ball_bearing, soft_close: true
  drawers 4, height: 140, slide: :ball_bearing

  # Рамочный фасад (как у дверей — рамка + филёнка)
  drawer 150, type: :frame
  drawers 2, height: 140, type: :frame
end
```

### Ряд ящиков (несколько в ряд)

Автоматически добавляет вертикальные перегородки между ящиками и горизонтальные полки между рядами.

```ruby
cabinet 800 do
  # Автоматический расчёт — count: делит ширину поровну
  drawer_row height: 150, count: 2    # два равных + перегородка
  drawer_row height: 200, count: 1    # один сплошной
  drawer_row height: 150, count: 3    # три равных + 2 перегородки
  
  # Явные ширины (для разных размеров)
  drawer_row height: 150 do
    drawer 400              # левый пошире
    drawer 346              # правый поуже
  end
end
```

Между ящиками в ряду — вертикальная перегородка.
Между рядами — горизонтальная полка (только если хотя бы один из смежных рядов имеет перегородки).

Фасады автоматически рассчитываются так, чтобы закрывать боковины корпуса, перегородки и горизонтальные полки. Зазор `facade_gap/2` на каждом краю корпуса и `facade_gap` между соседними фасадами.

Короб ящика меньше фасада по высоте и центрирован: `box_top_inset` (сверху) и `box_bottom_inset` (снизу) по умолчанию 20мм.

Параметры (`slide:`, `soft_close:`, `back_gap:`) наследуются от ряда, но могут быть переопределены для каждого ящика.

### Параметры ящиков

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `height` | Number | — | Высота ящика (мм) |
| `type` | Symbol | `:solid` | `:solid` — сплошной фасад, `:frame` — рамочный (рамка + филёнка) |
| `slide` | Symbol | `:ball_bearing` | Тип направляющих |
| `soft_close` | Boolean | `false` | Плавное закрывание |
| `draw_slides` | Boolean | `false` | Рисовать направляющие |
| `back_gap` | Number | `20` | Зазор до задней стенки шкафа (мм) |
| `box_top_inset` | Number | `20` | Отступ короба от верха фасада (мм) |
| `box_bottom_inset` | Number | `20` | Отступ короба от низа фасада (мм) |

Для рамочного фасада (`type: :frame`) доступны те же опции, что и у дверей: `frame_width`, `frame_thickness`, `tenon`, `panel_gap`, `panel_thickness`, `groove_depth`.

### Глобальные параметры фасадов

```ruby
SketchupFurniture.config.facade_gap = 3        # зазор между фасадами (мм)
SketchupFurniture.config.frame_thickness = 20  # толщина рамочного фасада (мм)
```

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `facade_gap` | Number | `3` | Зазор между фасадами ящиков (мм) |
| `frame_thickness` | Number | `20` | Толщина рамочного фасада по умолчанию (мм) |

## Двери

### Добавление дверей

```ruby
cabinet 600 do
  # Одна дверь (сплошная, ЛДСП)
  doors 1

  # Две створки (делят ширину пополам)
  doors 2

  # С параметром материала
  doors 1, facade_material: :mdf_19
  doors 2, facade_material: :mdf_19
end
```

Сторона петель рассчитывается автоматически:
- 1 дверь → петли слева
- 2 двери → левая влево, правая вправо

Фасады покрывают боковины шкафа с учётом `facade_gap`. Без фурнитуры (петли).

Двери можно открывать/закрывать двойным кликом (как ящики) через `activate_drawer_tool`.

### Рамочная дверь (type: :frame)

Рамка из массива с шипом и филёнка из фанеры в центре:

```
┌────────────────────────┐
│  ┌──────────────────┐  │
│  │                  │  │ ← стойки (стоевые)
│  │     филёнка      │  │
│  │     (фанера)     │  │
│  │                  │  │
│  └──────────────────┘  │
└────────────────────────┘
    ↑ поперечины (rails)
```

```ruby
cabinet 600 do
  # Рамочная дверь с параметрами по умолчанию
  doors 1, type: :frame

  # С настройками рамки
  doors 1, type: :frame, frame_width: 40, frame_thickness: 22

  # Две рамочные створки
  doors 2, type: :frame
end
```

В раскрой добавляется:
- **2 стойки** (массив): высота двери × frame_width × frame_thickness
- **2 поперечины** (массив): (внутренний проём + 2 × шип) × frame_width × frame_thickness
- **1 филёнка** (фанера): с учётом захода в паз и зазора panel_gap

Филёнка на `panel_gap` (2мм) меньше паза с каждой стороны.

### Параметры дверей

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `type` | Symbol | `:solid` | `:solid` — сплошная, `:frame` — рамочная |
| `facade_material` | Symbol | `:ldsp_16` | Материал фасада (для :solid) |
| `frame_width` | Number | `50` | Ширина бруска рамки (мм, для :frame) |
| `frame_thickness` | Number | `20` | Толщина рамки (мм, для :frame) |
| `tenon` | Number | `10` | Длина шипа (мм) |
| `panel_gap` | Number | `2` | Зазор филёнки от паза (мм с каждой стороны) |
| `panel_thickness` | Number | `6` | Толщина филёнки (мм) |
| `groove_depth` | Number | `= tenon` | Глубина паза (мм) |

### Анимация ящиков

```ruby
# Открыть/закрыть все
wardrobe.open_all_drawers
wardrobe.close_all_drawers

# Конкретный ящик (колонна, модуль, ящик)
wardrobe.open_drawer(1, 0, 0)      # колонна 1, модуль 0, ящик 0
wardrobe.close_drawer(1, 0, 0)

# Открыть частично (мм)
wardrobe.open_drawer(1, 0, 0, amount: 200)

# Получить все ящики
wardrobe.all_drawers              # массив объектов Drawer
```

## Типы опор

### Стандартный (на боковинах)

По умолчанию шкаф стоит на боковинах.

```ruby
base 450 do
  # боковины до пола
end
```

### Цоколь

Боковины до пола, но дно поднято — образуется ниша.

```ruby
base 450 do
  plinth 80                 # ниша 80мм
  plinth 80, front_panel: true  # с цокольной планкой
end
```

### Ножки

Боковины короче, шкаф на ножках.

```ruby
base 450 do
  legs 100                  # ножки 100мм
  legs 100, count: 6        # 6 ножек
end
```

## Пропуск частей

```ruby
cabinet 500 do
  skip :bottom          # без дна (для ящиков)
  skip :back            # без задника (открытое)
  skip :top             # без верхней крышки
  skip :left_side       # без левой боковины
  skip :right_side      # без правой боковины
  
  # Несколько сразу
  skip :bottom, :back
end
```

## Секции

```ruby
# Проценты (автоматический расчёт)
sections "50%", "50%"         # две равные
sections "30%", "70%"         # 30% и 70%
sections "33%", "34%", "33%"  # три секции

# Абсолютные значения (мм)
sections 400, 400, 328        # точные размеры
```

При изменении ширины колонны секции в процентах пересчитываются автоматически.

## API Reference

### Wardrobe

```ruby
Wardrobe.new(name, depth:, thickness: 18, &block)
```

| Метод | Описание |
|-------|----------|

### Cabinet

```ruby
cabinet(height, name:) do
  # Опоры
  plinth 80                   # цоколь
  legs 100                    # ножки
  
  # Секции и полки
  sections "50%", "50%"       # секции
  shelf 400                   # полка
  shelves [300, 600, 900]     # несколько полок
  
  # Ящики
  drawer 150                  # один ящик
  drawers 3, height: 140      # несколько
  drawers [0, 150, 350]       # по позициям Z
  
  # Ряд ящиков (несколько по горизонтали)
  drawer_row height: 150, count: 2  # авто
  drawer_row height: 150 do         # явные ширины
    drawer 382
    drawer 364
  end
  
  # Двери
  doors 1                      # одна дверь
  doors 2                      # две створки
  
  # Царги
  stretchers                  # стандартные
  stretchers :sink            # на ребро (под мойку)
  
  # Пропуск частей
  skip :bottom, :back         # без дна и задника
end
```
